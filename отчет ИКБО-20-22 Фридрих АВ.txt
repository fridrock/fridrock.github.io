ВВЕДЕНИЕ
     Целью данной работы является разработка информационного сайта по теме нереляционных СУБД.
     Для достижения этой цели необходимо решить ряд задач: 
     – провести анализ существующих решений в данной области:
     – определить потребностей аудитории сайта;
     – разработать структуру сайта, включающую в себя создание разделов, подразделов и страниц;
     – написать код и сверстать страницы;
     – обеспечить адаптивность сайта для различных устройств;
     – протестировать сайта. 
     Объектом исследования является веб-сайт по теме «Нереляционные СУБД».
     Предметом исследования выступают процессы разработки, создания и функционирования веб-сайта «Нереляционные СУБД».
     Методы исследования: анализ научно-методической литературы и аналогичных веб-сайтов, синтез, моделирование.


     1 ОБЩИЕ СВЕДЕНИЯ
     1.1 Наименование клиентской части интернет-ресурса
     Клиентская часть интернет-ресурса «Нереляционные СУБД» получила название «NOSQLLearn». Это название отражает ключевую функцию клиентской части – обеспечение полной и верной информации о нереляционных СУБД.
     1.2 Функциональное назначение
     Веб-сайт «NOSQLLearn» обладает набором функций для просмотра основной страницы с навигацией и информации по различным курсам, чтения статей по теме, решения упражнений и написания обратной связи. Все это предназначено для удовлетворения потребностей пользователей в получении информации о нереляционных СУБД.
     1.3 Прикладное программное обеспечение
     Для работы информационного сайта требуется следующее программное обеспечение: редактор кода Visual Studio Code и веб-сервис GitHub Pages.
